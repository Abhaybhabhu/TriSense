# TriSense

TriSense is a low-cost wearable sensor system for triathlon technique feedback. The first prototype focuses on swimming, using a waterproof IMU-based pod to record body movement data for post-session analysis.

## Prototype 0.1 Aim

The first prototype aims to prove that a small wearable sensor pod can:

- record acceleration and gyroscope data
- save timestamped motion data to a microSD card
- be worn on the lower back during swim-like movement
- produce useful post-session plots such as body-roll and stroke-rhythm trends

## Initial Hardware

- Seeed Studio XIAO nRF52840 Sense
- microSD module
- 3.7V 1S LiPo battery
- microSD card
- waterproof enclosure
- lower-back strap or tri-suit pocket mounting

## Planned Development Stages

1. IMU serial output test
2. microSD write test
3. IMU data logging to CSV
4. battery-powered test
5. lower-back dry movement test
6. waterproof enclosure test
7. swim or swim-like validation test
